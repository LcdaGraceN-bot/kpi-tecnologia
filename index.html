Configuración

Conecta Google Sheets para sincronizar datos entre el equipo

¿Por qué Google Sheets? Es gratuito, todos en el equipo pueden ver el mismo dashboard en tiempo real, y no necesitas servidor. Solo necesitas una cuenta Google institucional y seguir los pasos de abajo.

Guía de conexión · Google Apps Script

  1. Crea una Google Sheet nueva en tu cuenta institucional. Nómbrala KPI Tecnología. En la primera fila escribe exactamente estas cabeceras en columnas A–R:
    timestamp | anio | periodo | responsable | k1_plan | k1_impl | k1_pct | k1_ev | k2_total | k2_cap | k2_pct | k2_ev | k3_total | k3_uso | k3_pct | k3_plat | k3_avg | k3_ev | k4_conc | k4_men | k4_nombres | k4_ev | obs
  2. En la Sheet ve a Extensiones → Apps Script. Borra el código por defecto y pega el siguiente: 
    function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet()
              .getSheetByName("Hoja 1");
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow(data.row);
  return ContentService
    .createTextOutput(JSON.stringify({status:"ok"}))
    .setMimeType(ContentService.MimeType.JSON);
}
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet()
              .getSheetByName("Hoja 1");
  var rows = sheet.getDataRange().getValues();
  return ContentService
    .createTextOutput(JSON.stringify(rows))
    .setMimeType(ContentService.MimeType.JSON);
}
  3. Haz clic en Implementar → Nueva implementación. Tipo: Aplicación web. Ejecutar como: Yo. Acceso: Cualquier persona. Copia la URL que te entrega.
  4. Pega esa URL en el campo de abajo y guarda.
